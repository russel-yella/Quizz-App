# Quiz App 🎮

A modern Python quiz application with **Tkinter GUI**, a **countdown timer**, and **True/False questions**.  
Built to practice Python OOP, GUIs, and interactive design.

---

## Features ✨

- Modern, clean UI with card-style question layout  
- Countdown timer for each question (10 seconds)  
- Score tracking with visual feedback  
- True/False questions  
- Easy to extend to multiple choice or categories  
- Works with any question dataset following a simple structure  

---

## Installation 🛠️

1. Clone this repository:

```bash
git clone https://github.com/yourusername/quiz-app.git
cd quiz-app
```
2.Make sure you have Python 3.x installed.
3.Install required packages (Tkinter is included in most Python distributions):
```bash
pip install -r requirements.txt
```
